## Top Findings (Structured)

### F1: Hallmark (Nutlope/hallmark) — Very High Novelty
- 65+ blocking slop-test gates + pre-emit self-critique + 22 themes + 4 verbs (build/audit/redesign/study).
- "Study" extracts design DNA from screenshots/URLs without cloning.
- Clean `npx skills add` distribution and multi-env support.
- **Relevance:** Direct pattern for `simplicity-guard` enhancement and new output-quality rows in shipped protocols.

### F2: 13-Agent Neurosymbolic System (zoharbabin) — Highest ROI
- 38-step resumable pipeline, 9 specialists + 4 meta-agents.
- **Deterministic symbolic cross-domain trigger rules** (Python after LLM pass-1).
- 5 **hard blocking** quality gates: coverage, schema, citation verification (exact page/quote), semantic dedup, numerical audit.
- Persistent compound KB with NetworkX graph + SHA-256 fingerprint lineage + chronicle across runs.
- "NOT_FOUND" escape + accusatory framing for self-verification.
- **Relevance to nullprobe:** Almost 1:1 with what Exploration v3, verification/audit protocols, and wiki/Cognitive Firewall need.

### F3: Hindsight / Institutional Memory (Vectorize)
- Explicit split: Personalization vs **Institutional Knowledge** (the harder problem).
- Fact extraction + entity resolution at write time.
- 4 parallel retrieval strategies + rerank + `reflect` (LLM synthesis).
- MCP-first, embedded PG self-host.
- **Relevance:** Model for evolving nullprobe's wiki from append-only log to true compounding institutional memory.

### F4: AIDD + SudoLang
- Large library of specialized reusable SKILL.md files.
- SudoLang: constraint-based typed pseudocode (20-30% fewer tokens, better determinism on complex flows).
- vision.md as source of truth + AGENTS.md custom/ override import pattern.
- **Relevance:** Selective pattern borrowing (SudoLang for prompts, churn analysis, custom override UX).

### F5: Agent Skills Standard (agentskills.io, 2026)
- (Discovered during broader sweep) The 2025-2026 open standard (SKILL.md + YAML frontmatter + progressive disclosure) that Hallmark, AIDD, and many others follow.
- Validates nullprobe's own multi-environment skill shipping approach.
- **Relevance:** Alignment opportunity + credibility.
